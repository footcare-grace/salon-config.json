SNS集客プロンプト・アトリエ

フットケアサロン
集客支援ツール

サロンの基本情報を一度入力すれば、リサーチ／コンテンツ制作／マーケティング／実行計画の 4種のAIプロンプトが自動で組み上がります。生成された文章をコピーして、Claude や Gemini に貼り付けるだけ。 外部への送信は一切行いません。

基本情報サロン情報の入力

初回は salon-config.json の内容を読み込みます。ここで上書きした内容はこのブラウザにのみ一時保存され、 4つのプロンプトへ即時反映されます。

  1. 一リサーチ
  2. 二コンテンツ制作
  3. 三マーケティング
  4. 四実行計画

①の出力を②③へ、①〜③の出力を④へ貼り付けて使う、リレー形式の設計です。

リサーチチーム用プロンプト

ペルソナ設計・悩みの言語化・地域競合分析を一度に依頼します。最初にこのプロンプトを実行してください。


      

    


    

      

        

          
コンテンツ制作チーム用プロンプト

          
①で得たペルソナ・差別化情報を前提に、Threads/note投稿案とリール構成案を週次カレンダー形式で生成します。①の回答は「④統合ディレクター」タブの入力欄に貼り付けると、ここへ自動反映されます。

        

        

          
          
        

        

      

    


    

      

        

          
マーケティングチーム用プロンプト

          
SNS→公式LINE→予約のファネル設計、特典設計、初回予約のハードルを下げる施策を立案させます。①の回答は「④統合ディレクター」タブの入力欄に貼り付けると、ここへ自動反映されます。
